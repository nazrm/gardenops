GardenOps vendors these font files so the frontend does not fetch web fonts at
runtime.

- `OpenSans-*.ttf`: copied from the host `open-sans-fonts` RPM and used for
  the app UI.
- `DejaVuSansCondensed-Bold.ttf`: copied from the host `dejavu-sans-fonts` RPM
  and used for compact map labels such as `P1` and `S1`.

Keep the matching license files in this directory when updating these assets.
